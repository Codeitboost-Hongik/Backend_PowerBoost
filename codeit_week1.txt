터미널 사용법
   man [명령어] : 명령어에 대한 메뉴얼을 볼 수 있음
   커맨드 작성 중 커맨드 작성 취소 : ctrl+c
   터미널 내용 지우기 : clear
      =>내용은 지워지지만 커맨드history는 남아있음

파일경로
    루트디렉토리부터 파일까지의 경로
    디렉토리안으로 들어갈경우' / ' 로 구분
    pwd : 현재 위치한 디렉토리 경로 출력
    cd [argument] :  다른 디렉토리로 이동
    cd / :root디렉토리로 이동
    cd ~ : home디렉토리로 이동
    cd - :이전에 있던 디렉토리로 이동
    ls [argument] : argument는 경로 값으로, 해당 디렉토리의 있는 내용물을 보여줌
                    디폴트값은 현재 디렉토리의 내용물을 리스트형식으로 보여줌
    상위디렉토리로 이동시 : ' .. ' 으로 이동    


mkdir [argument] : 디렉토리 생성
touch [argument] : 파일생성
cat [argument]: 파일의 내용을 보여줌
mv [argument1] [argument2] : argument1은 작업할 대상의 경로, argument2는 이동할 목적지 또는 변경할 이름
                             argument2 : 이미 존재하는 디렉토리의 경로일 경우 해당 디렉토리 안으로 파일을 옮기고, 그렇지 않으면 이름이 변경됨.
cp [argument1] [argument2] : argument1은 복사할 대상의 경로, argument2는 복사할 위치
                             argument2 : 이미 존재하는 디렉토리라면 그 안으로 복사되고, 아니면 argument2라는 파일 또는 디렉토리가 생성                       
rm [argument] : argument는 삭제할 파일                              
rm -r [argument] : 옵션 -r 은 무조건 붙여야함. argument는 삭제할  디렉토리 경로


Vim
    CLI 환경에서 파일 내용 작성하는 tool
    vim 모드
      1.일반모드 : 커서이동, 텍스트 붙여넣기, 작업취소
      2.입력모드 : 텍스트 입력
      3.비주얼모드 : 텍스트 블록 지정, 텍스트 복사
      4.명령모드 : 내용저장, vim종료
      vim을 처음열면 일반모드로 실행됨
      다른 모드로 가려면 항상 일반모드로 거쳐야됨
      입력모드로 전환 : i
      비주얼모드로 전환 : v 혹은 V
      명령모드로 전환 : ' : '
      일반모드로 전환 : esc 키
